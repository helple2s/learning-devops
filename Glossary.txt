pwd (Print Working Directory) — «Де я?». 

ls — список файлів.

   ls -la — показує все, включаючи приховані файли

cd (Change Directory) — перехід між папками.

   cd .. — вийти на рівень вгору.

   cd ~ — повернутися в «домашню» папку.

touch <назва> — створити порожній файл.

mkdir <назва> — створити папку.

cp (Copy) — копіювати.

mv (Move) — перемістити або перейменувати.

rm (Remove) — видалити.

echo "текст" — вивести текст у термінал.

   echo "текст" > file.txt — записати текст у файл (старе видалиться).

   echo "текст" >> file.txt — додати текст у кінець файлу.

cat — вивести весь вміст файлу на екран.

less — відкрити файл для читання 

head / tail — показати перші або останні 10 рядків файлу.

   tail -f log.txt — показує нові рядки у файлі в реальному часі. Ідеально для стеження за     логами.

sudo (SuperUser Do) — виконати команду від імені адміна.

man <команда> (Manual) — інструкція до будь-якої команди.

clear — очистити екран терміналу
